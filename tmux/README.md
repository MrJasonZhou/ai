# tmux ツール & 設定集

本ディレクトリには、tmux での作業効率を向上させるためのスクリプトおよびステータスバー設定方法が含まれています。

---

## 🚀 1. tmux セッションインタラクティブ選択ツール (`ta`)

### 概要
Python の標準ライブラリ (`curses`) のみを使用した、tmux セッション選択メニューです。
ターミナル上で現在起動している tmux セッションの一覧を表示し、上下キーで選択してアタッチ（移行）できます。新規セッションの作成も選択可能です。

### 導入方法
1. `ta` スクリプトを実行パス（例: `~/.local/bin` や `/usr/local/bin`）にコピーします。
   ```bash
   cp ta ~/.local/bin/ta
   ```
2. 実行権限を付与します。
   ```bash
   chmod +x ~/.local/bin/ta
   ```

### 使い方
ターミナル上で以下のコマンドを実行するだけです。
```bash
ta
```
*   **操作方法:**
    *   `↑` / `↓` (矢印キー): セッションの選択移動
    *   `Enter`: 選択したセッションへアタッチ (新規セッション作成オプションを選択した場合は、新規 tmux セッションが立ち上がります)

## 📊 2. tmux ステータスバーへの API Token 使用量表示

### 概要
tmux の画面下部にあるステータスバー（`status-right`）に、AI アシスタント（`agy` など）の API トークン使用量を表示する設定です。

### 特徴
*   **インテリジェント表示:** AI アシスタント (`agy` など) がアクティブに動作しているときのみトークン使用量を表示し、シェル（bash等）に退出しているときや他のセッションでは自動的に非表示になります。
*   **時間枠別の使用率:** 5時間リミットおよび週間リミットに対する割合（%）を表示します。

### 設定方法
使用しているトークン計算スクリプト（例: `calc_token_usage.py`）のパスを調べて、以下のコマンドをターミナルで実行するか、`~/.tmux.conf` に追加します。

#### コマンドラインで一時的に適用する場合:
```bash
tmux set -g status-right "#(python3 /path/to/calc_token_usage.py --tmux) | \"#{=21:pane_title}\" %H:%M"
```

#### `~/.tmux.conf` に永続化する場合:
```tmux
set -g status-right "#(python3 /path/to/calc_token_usage.py --tmux) | \"#{=21:pane_title}\" %H:%M"
```
